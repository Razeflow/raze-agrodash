github.com/Razeflow/raze-agrodash

What It Is

A real-time agricultural monitoring portal for the Municipality of Tubo. It lets barangay encoders submit farming records, while municipal admins see live analytics — commodity production, farmer counts, damage reports, program benefits, and barangay performance rankings — all in one dashboard.

Modules & Features

All Users

Overview

  • KPI cards — total farmers, production, planting area, damage, top commodity
  • Finding Matrix — commodity status per barangay (Productive / At Risk / Damaged / No Activity)
  • Commodity Analytics — bar chart + pie chart of production by commodity
  • Sub-Category Breakdown — variety-level production per commodity
  • Daily Summary Calendar — monthly / weekly / daily views of record activity
  • Barangay Leaderboard — ranked by production, farmers, area, or calamity damage (admin only)
  • Date range filter + barangay filter applied across all sections
All Users

Damage & Risk Monitoring

  • Weather alert feed and loss data
  • Calamity damage and pest damage tracking by hectare
  • Risk classification per barangay and commodity
All Users

Farmers

  • Full municipal farmer database
  • Farmer profiles — name, gender, barangay, contact
  • Add / Edit / Delete farmers
  • Barangay and gender distribution charts
All Users

Records

  • Production log per commodity and sub-category
  • Compact card layout with quick stats per record
  • Filters: search, commodity, barangay, period (month/year), date range
  • Add / Edit / Delete records
  • Fishery records support separate harvest fields
  • Analytics panel — record count, top commodity, total planting area
All Users

Programs

  • RFFA (Registry of Fisherfolk and Farmers) tracking
  • Subsidy and benefit distribution records
  • Farmer organization membership
  • Government program enrollment per barangay
Admin Only

Management

  • Barangay portal user credentials grid
  • Shows username + active/unassigned status per barangay
  • Inline password reset for any barangay user
  • Passwords are hashed — plaintext never displayed
Admin Only

Users

  • Full admin role management
  • Create / edit / delete admin and barangay accounts
  • Assign roles: Super Admin, Admin, Barangay User
  • Assign barangay to each user account
All Users

Export & Account

  • Export button — downloads filtered records as CSV/Excel
  • Change password dialog (available to all roles)
  • Session-based login with role auto-redirect
